# Ultra-Fighters

A 3D Super Fighters Remake

[The Original that Inspired Me](http://www.twoplayergames.org/play/592-Super_Fighters.html)


### Controls

To move: WASD (Easy, right?) <br>
To Look around: Arrow keys (All keyboard, huh?)
